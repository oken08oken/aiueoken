すきるについて

  • HTML

    マークアップはサイトの要。構造も考えながらしっかりとしたマークアップが出来るようにしていきたい。最近ではアクセシビリティの重要性が高まってきいるが正直手を出せていないので、これから学びたい。

  • CSS

    最近になって、時間を最も費やしているのがこの言語。特にGridレイアウトはこれから超重要になってくると思うので使いこなせるようになりたい。CSSは嫌いな人が多いらしいので、頑張れば最も恩恵が大きいのではと期待。

  • JavaScript

    一時期頑張って学んでいました。けっこう前にブラウザで遊べるすごろくとかテトリスをアレンジしたミニゲームを作りましたが最近はめっきりです。web制作だとDOM操作とかがメインになってくると思うので現状では優先順位が低めです。

  • GSAP

    JS自体は最近全くでしたがこちらに関しては結構学んでいて、そのうちがっつり使いたいと思っています。プラグインも豊富ですが、コアとなるタイムライン機能やイージングなどが便利なので使いこなせるようになりたい。

  • WordPress

    正直ほとんど使ったことないので分からないです。プラグインの更新が頻繁に求められるのは知っていますが、ほぼ

  • デザイン

    最も避けてきた分野になります。できればやりたくないと思っていましたが、やっぱり自分でかっこいいデザインを作れるようになりたいと思っています。色々なものを見て考える力をつけたい。
